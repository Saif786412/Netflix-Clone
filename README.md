# Netflix Clone Cloud Security Project

A cloud-enabled Netflix Clone developed as an BCA Major Project.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Firebase Authentication
- Cloud Firestore
- Firebase Hosting

## Security Features
- Role-Based Access Control (RBAC)
- Route Protection
- Session Persistence
- Input Sanitization
- Security Event Logging
- Firestore Security Rules
- Administrative Session Verification

## Functional Features
- User Registration
- User Authentication
- Member Homepage
- Hero Banner Management
- Movie Catalogue Management
- Admin Dashboard
- Responsive User Interface
